Recently, however, Javascript has started to make a pull outside of then web browser.
In 2009 Node.js was released.

Node.js enables Javascript on the serverside. This framework allows developers to develop in just one langauge if they so choose, Javascript.
http://tagpro.gg is a great example of a game which is written entirely in Javascript, clientside and serverside.
Node.js introduced the notion that Javascript can be used for more than just purpose.
Javascript began popping up in a slew of new locations.


Leap Motion


A sensor hardware called Leap Motion uses a Javascript API for communication. The Leap Motion measures distances above of it. Javascript is used to control the receptors on the Leap Motion. This is actual hardware which Javascript is controlling. Through tools like the Leap Motion, Javascript is moving into a role of a programming language across platforms, outside of the web browser.


pijs.io

pijs.io is another great example of how Javascript can be easily harnessed across platforms. pijs.io is a Javascript framework that enables programmers to interface directly with a Raspberry PI, a fairly basic computer. This means Javascript is being used to maintain system calls in a machine.

With these current trends in widespread usage of Javascript, the future will bring about even more Javascript utility.
